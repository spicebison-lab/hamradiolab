How it's calculated: Matched loss is computed from published attenuation coefficients interpolated for your frequency using the cable's α·√f + β·f model. SWR additional loss adds the mismatch penalty:
SWR loss (dB) = matched_dB + 10·log₁₀( (1 + Γ²·10^(-matched_dB/5)) / (1 - Γ²) )
where Γ = (SWR−1)/(SWR+1). Power loss % = 1 − 10^(−dB/10). Results assume standard temperature (20°C / 68°F).
